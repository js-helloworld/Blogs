# 20.有效的括号

**难度：容易**

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

**示例 1：**

```
输入：s = "()"
输出：true
```

**示例 2：**

```
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```
输入：s = "(]"
输出：false
```

**提示：**

- `1 <= s.length <= 104`
- `s` 仅由括号 `'()[]{}'` 组成

```javascript
/*
 * @lc app=leetcode.cn id=20 lang=javascript
 *
 * [20] 有效的括号
 */

// @lc code=start
/**
 * @param {string} s
 * @return {boolean}
 */
var isValid = function(s) {
  const keyMap = {
    '(':-3,
    '{':-2,
    '[':-1,
    ']':1,
    '}':2,
    ')':3,
  }
  const len = s.length;
  if(keyMap[s[0]] > 0 || len % 2 === 1) return false;
  const stack = [s[0]];
  for(let i = 1; i < len; i++) {
    if (keyMap[stack[stack.length-1]] + keyMap[s[i]] === 0) {
      stack.pop();
    }else if(keyMap[s[i]] > 0) {
      return false
    }else {
      stack.push(s[i])
    }
  }
  return stack.length === 0;

};
// @lc code=end


```
